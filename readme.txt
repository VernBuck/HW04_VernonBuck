This is a C implementation of Breakout, the movement controls are the GBA directional controls (left and right arrow)
If you destroy all of the bricks then the screen will trun gold and say that you win. If the ball that erases blocks
hits the bottom of the screen then you will lose, to play again press start.To pause midgame press the start button. 
To start the game on the splash screen press the start button. During a pause, if you press the select button you
return to the splash screen.
